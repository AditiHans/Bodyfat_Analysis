The Body Fat Analysis System is a data-driven predictive model designed to estimate body fat percentage using anthropometric and physiological measurements.
By combining regression modeling, dimensionality reduction, and clustering, the project demonstrates how statistical and machine learning techniques can uncover relationships between body metrics and composition patterns.
